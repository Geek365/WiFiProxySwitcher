# 功能
* 一键增加 WLAN 高级设置中的 Http Proxy 代理设置
* 一键取消此代理设置
* 兼容 4.0 ~ 6.0 系统

# 注意事项
* 6.0 系统需要在系统设置断开 WiFi, 单独在软件中连接 WiFi
* 需要配合代理主机使用
